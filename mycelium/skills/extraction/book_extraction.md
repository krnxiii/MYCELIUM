# Book Extraction Skill

## Focus
Extract author, key thesis, chapter structure, and core concepts.

## Neuron Types
- concept: main ideas, frameworks, mental models
- person: authors, cited figures
- belief: author's core claims
- resource: books, papers cited

## Synapse Patterns
- author CREATED book
- book ARGUES_FOR thesis
- concept INFLUENCES concept
