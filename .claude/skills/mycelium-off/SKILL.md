---
name: mycelium-off
description: Disable MYCELIUM knowledge graph access. Use when user says "disable mycelium", "turn off mycelium", "mycelium off", or wants to stop graph access.
---

Disable MYCELIUM knowledge graph access.

Steps:
1. Run: `rm -f ~/.mycelium/.read_enabled ~/.mycelium/.write_enabled`
2. Confirm to user: "MYCELIUM access disabled."
